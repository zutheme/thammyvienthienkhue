== Description ==

Giúp khách có thể dùng Zalo để chat với bạn ngay khi ở trên web
- Tin nhắn vào Zalo trên điện thoại của bạn ngay
- Gởi ảnh, đính kèm tập tin dễ dàng
- Gởi biểu tượng cảm xúc ngộ nghĩnh
- Lưu lại kết nối với khách hàng dễ hơn

Để tích hợp bạn cần tài khoản Zalo Official Account. Bạn có thể tạo theo link sau: https://oa.zalo.me/manage/oa?option=create
Sau đó vào Setting -> Zalo Live Chat và điền mã tài khoản vào
== Installation ==


1. Cài đặt Plugin
2. Setting -> Zalo Live Chat
3. Điền thông tin Zalo Official Account ID
